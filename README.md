# Service Connection Platform (Angular)

Standalone Angular frontend with Tailwind CSS and mock data only. Backend integration will be added later.

## Requirements

- Node.js 20.19+ (or 22+)

## Install and run

```bash
npm install
npm run start
```

The app runs at http://localhost:4200.

## Routes

- /role
- /admin/*
- /customer/*
- /provider/*

## Notes

- All data is mocked in src/app/core/mock-data.ts.
- UI is built with Tailwind v3 and custom design tokens in tailwind.config.js.
